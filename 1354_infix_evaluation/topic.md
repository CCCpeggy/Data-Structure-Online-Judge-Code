# 1354: 108資料結構期中考 - Infix Evaluation

* Time Limit: 1 Sec
* Memory Limit: 128 MB

## Description

``` text
Please develop a program which can evaluate Infix Expression. For example, input an expression represented by a string 1+(1+2)*4-6/3, then output the result of this expression.

Assume +, -, *, and / binary operations are allowed only.

Operator associativity is from left to right, and the defined operator precedence is listed below.
```

|Operator|Precedence|
|:---:|:---:|
|()|1|
|* 、 /|2|
|+ 、 -|3|

``` markdown
See the Sample Input.
```

## Output

``` markdown
See the Sample Output.
```

## Sample Input

``` markdown
4+(2+1)*6+(2+2*3+1)/(1+2)-5
(((200-(1+1)*(90+(30/3))))/10)
```

## Sample Output

``` markdown
20
0
```
